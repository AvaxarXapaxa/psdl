Welcome to psdl's documentation!
================================

PSDL is a pythonic wrapper around the `SDL Library`_

PSDL helps you to make multimedia grahical applications (like games) using
python!

Take a look at the installation guides first, to get psdl up and running,
then you can dive straight into the docs.
